# Providers
Anify uses multiple providers as it's sources. The following list are providers that Anify uses.

## Anime

[AnimeFox:](https://animefox.tv) Looks & functions similarly to Zoro.

[Animepahe:](https://animepahe.com) Often provides high quality anime streaming links.

[Crunchyroll:](https://crunchyroll.com) Unfortunately not a very functioning provider, but the code exists if you want to use it.

[Enime:](https://api.enime.moe) An API that caches anime episodes & their publicly-available, third-party sources.

[GogoAnime:](https://gogoanime.dk) The default provider for anime - one of the more well-known/established providers.

[Zoro:](https://zoro.to) A well-known site named after Roronoa Zoro from ONE PIECE.

## Manga

[Mangadex:](https://mangadex.org) Hosts 10,000s of chapters of scanlated manga. Generally very high-quality (doesn't compress images).

[Mangakakalot:](https://mangakakalot.com) Read manga online in English. Contains a larger library of manga.

[Mangapark (v2):](https://v2.mangapark.net) The second version of Mangapark - a large collation of manga available for free, in English.

[Mangasee123:](https://mangasee123.com) The default provider for manga - a large library of over 6,000 manga available in English.

## Meta

[Anilist:](https://anilist.co) A metadata provider used to aggregate data from Anilist and map information to other providers.

[The Movie Database (TMDB):](https://www.themoviedb.org) Used for banners/images and information mapping.
