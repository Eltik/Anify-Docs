# Schedule
Used for querying AniList's upcoming schedule route and returning media. Technical details regarding the usage of the schedule function can be found below. Example code is provided for both JavaScript and Python, along with a response schema.

## Rate Limits
The schedule route has a rate limit of 60 requests per minute unless your API key is whitelisted.

        <Tabs items={["JavaScript", "Python"]}>

                <>
                    ```js
                    import dotenv from "dotenv";
                    dotenv.config();

                    import axios from "axios";

                    const apiKey = process.env.API_KEY ?? "";
                    const url = `https://api.anify.tv/schedule?apikey=${apiKey}`;
                    const data = async () => {
                        try {
                            const { data } = await axios.get(url);
                            return data;
                        } catch (err) {
                            throw new Error(err.message);
                        }
                    };

                    console.log(data);
                    ```
                </>
            </Tab>

                    response = requests.get(url)
