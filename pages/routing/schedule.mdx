# Schedule
**This route is not stable**. Used for querying AniList's upcoming schedule route and returning media. Technical details regarding the usage of the schedule function can be found below. Example code is provided for both JavaScript and Python, along with a response schema.

## Rate Limits
The schedule route has a rate limit of 60 requests per minute unless your API key is whitelisted.

## Route Schema (URL)
<Tabs items={["Query", "Post"]}>
    <Tab>
        <>
            ```
            https://api.anify.tv/schedule?type={type}
            ```
        </>
| Parameter |  Type  | Description                                                       | Required? | Default |
| :-------: | :----: | ----------------------------------------------------------------- | :-------: | :-----: |
|   type    | string | The type of media to search for. Must be either `anime`, `manga`, or `novel`. |    Yes    |  `""`   |
|   fields  | string[]| The fields to return in the response. For example, if you want the title and cover image, you would input `title,coverImage`. |    No    |  `[]`   |
|   apikey  | string | Whitelisted API key |    No    |  `""`   |

        ## Request Samples

        <Tabs items={["JavaScript", "Python"]}>
            <Tab>
                <>
                    ```js
                    import axios from "axios";

                    const url = `https://api.anify.tv/schedule`;
                    const data = async () => {
                        try {
                            const { data } = await axios.post(url, {
                                type: "anime"
                            });
                            return data;
                        } catch (err) {
                            throw new Error(err.message);
                        }
                    };

                    console.log(data);
                    ```
                </>
            </Tab>
            <Tab>
                <>
                    ```python
                    import requests
                    
                    url = "https://api.anify.tv/schedule"
                    response = requests.post(url, json={'type': "anime"})
                    data = response.json()

                    print(data)
                    ```
                </>
            </Tab>
        </Tabs>
    </Tab>
</Tabs>
