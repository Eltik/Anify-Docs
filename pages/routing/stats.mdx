# Stats
Used for getting the number of anime, manga, novels, api keys, and skip times stored in the database. Technical details regarding the usage of the stats function can be found below. Example code is provided for both JavaScript and Python, along with a response schema.

## Rate Limits
The stats route has a rate limit of 60 requests per minute unless you provide a whitelisted API key.
