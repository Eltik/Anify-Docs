# Search
Used for querying media. This route will add media to the mapping queue to be mapped if there are no results found (which is why a high rate limit is necessary). Technical details regarding the usage of the search function for anime, manga, and novels can be found below. Example code is provided for both JavaScript and Python, along with a response schema.

## Rate Limits
The search route has a rate limit of 25 requests per minute unless you provide a whitelisted API key.

## Route Schema (URL)
<Tabs items={["Parameters", "Query", "Post"]}>
    <Tab>
        <>
            ```
            https://api.anify.tv/search/{type}/{query}/{sort}/{sortDirection}
            ```
        </>
| Parameter |  Type  | Description                                                       | Required? | Default |
| :-------: | :----: | ----------------------------------------------------------------- | :-------: | :-----: |
|   query   | string | The search query; i.e. the title of the item you are looking for. |    Yes    |  `""`   |
|   type    | string | The type of media to search for. Must be either `anime`, `manga`, or `novel`. |    Yes    |  `""`   |
|   sort    | string | How to sort the returned data. Must be either `averageRating`, `averagePopularity`, `title->>'english'` (hopefully will be changed), `year`, `totalEpisodes`, `totalChapters`, or `totalVolumes`. |    No    |  `averageRating`   |
|   sortDirection    | string | Which direction to sort. Must be either `ASC` or `DESC`. |    No    |  `ASC`   |
|   apikey  | string | Whitelisted API key |    No    |  `""`   |

        ## Request Samples

        <Tabs items={["JavaScript", "Python"]}>
            <Tab>
                <>
                    ```js
                    import axios from "axios";

                    // Using the example query "mushoku tensei"
                    ```
                    ```js
                    const url = `https://api.anify.tv/search`;
                    const data = async () => {
                        try {
                            const { data } = await axios.post(url, {
                                type: "anime",
                                query: "mushoku tensei",
                                sort: "averagePopularity",
                                sortDirection: "ASC"
                            });
                            return data;
                        } catch (err) {
                            throw new Error(err.message);
                        }
                    };

                    console.log(data);
                    ```
                </>
            </Tab>
            <Tab>
                <>

                    ```python
                    # Using the example query "mushoku tensei"
                    url = "https://api.anify.tv/search"
                    response = requests.post(url, json={'type': "anime", 'query': "mushoku tensei", 'sort': "averagePopularity", 'sortDirection': "ASC"})
                    data = response.json()

                    print(data)
                    ```
                </>
            </Tab>
