# Chapters (MAL)
Used for fetching chapters for a given media but using MAL IDs rather than AniList IDs. Technical details regarding the usage of the chapters MAL function can be found below. Example code is provided for both JavaScript and Python, along with a response schema.

## Rate Limits
The episodes MAL route has a rate limit of 40 requests per minute unless your API key is whitelisted.

        ## Request Samples

        <Tabs items={["JavaScript", "Python"]}>
            <Tab>
                <>
                    ```js
                    import dotenv from "dotenv";
                    dotenv.config();

                    import axios from "axios";

                    const apiKey = process.env.API_KEY ?? "";
                    const url = `https://api.anify.tv/chapters-mal?apikey=${apiKey}`;
                    const data = async () => {
                        try {
                            const { data } = await axios.post(url, {
                                id: "147171"
                            });
                            return data;
                        } catch (err) {
                            throw new Error(err.message);
                        }
                    };

                    console.log(data);
                    ```
                </>
            </Tab>
            <Tab>
                <>
                    ```python
                    import requests
                    import os
                    from dotenv import load_dotenv

                    load_dotenv()

                    os.getenv('API_KEY')

                    url = "https://api.anify.tv/chapters-mal?apikey={}".format(os.getenv('API_KEY'))
                    response = requests.post(url, json={'id': "147171"})
                    data = response.json()

                    print(data)
                    ```
                </>
