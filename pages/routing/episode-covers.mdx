# Episode Covers
Used for querying episode covers for a media. Technical details regarding the usage of the episode covers function for anime can be found below. Example code is provided for both JavaScript and Python, along with a response schema.

## Rate Limits
The episode covers route has a rate limit of 40 requests per minute unless your API key is whitelisted.

        ## Request Samples

        <Tabs items={["JavaScript", "Python"]}>
            <Tab>
                <>
                    ```js
                    import dotenv from "dotenv";
                    dotenv.config();

                    import axios from "axios";

                    const apiKey = process.env.API_KEY ?? "";
                    const url = `https://api.anify.tv/episode-covers?apikey=${apiKey}`;
                    const data = async () => {
                        try {
                            const { data } = await axios.post(url, {
                                id: "108465"
                            });
                            return data;
                        } catch (err) {
                            throw new Error(err.message);
                        }
                    };

                    console.log(data);
                    ```
                </>
            </Tab>
            <Tab>

                    response = requests.post(url, json={'id': "108465"})
                    data = response.json()

                    print(data)
