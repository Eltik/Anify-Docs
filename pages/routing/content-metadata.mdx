# Content Metadata
Used for querying content metadata such as episode covers, episode description, chapter info, etc. for a media. Technical details regarding the usage of the content metadata function can be found below. Example code is provided for both JavaScript and Python, along with a response schema.

## Rate Limits
The content metadata route has a rate limit of 50 requests per minute unless your API key is whitelisted.

                    import axios from "axios";
