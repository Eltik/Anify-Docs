# Skip Times
Used for querying the skip times for a media. Technical details regarding the usage of the skip times function for anime can be found below. Example code is provided for both JavaScript and Python, along with a response schema.

## Rate Limits
The skip times route doesn't have a current rate limit.

## Route Schema (URL)

        ## Request Samples

        <Tabs items={["JavaScript", "Python"]}>
            <Tab>
                <>
                    ```js
                    import dotenv from "dotenv";
                    dotenv.config();

                    import axios from "axios";

                    const apiKey = process.env.API_KEY ?? "";
                    const url = `https://api.anify.tv/skip-times?apikey=${apiKey}`;
                    const data = async () => {
                        try {
                            const { data } = await axios.post(url, {
                                id: "108465"
                            });
                            return data;
                        } catch (err) {
                            throw new Error(err.message);
                        }
                    };

                    console.log(data);
                    ```
                </>
            </Tab>
            <Tab>
                <>
                    ```python
                    import requests
                    import os
                    from dotenv import load_dotenv

                    load_dotenv()

                    os.getenv('API_KEY')

                    url = "https://api.anify.tv/skip-times?apikey={}".format(os.getenv('API_KEY'))
                    response = requests.post(url, json={'id': "108465"})
                    data = response.json()

                    print(data)
                    ```
                </>
            </Tab>
        </Tabs>
