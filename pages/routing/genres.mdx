# Genres
Used for returning all unique genres in the database that can be then used for the `search-advanced` route. Technical details regarding the usage of the stats function can be found below. Example code is provided for both JavaScript and Python, along with a response schema.

## Rate Limits
The genres route doesn't have a current rate limit.

        ## Request Samples
        <Tabs items={["JavaScript", "Python"]}>
            <Tab>
                <>
                    ```js
                    import dotenv from "dotenv";
                    dotenv.config();

                    import axios from "axios";

                    const apiKey = process.env.API_KEY ?? "";
                    const url = `https://api.anify.tv/genres?apikey=${apiKey}`;
                    const data = async () => {
                        try {
                            const { data } = await axios.get(url);
                            return data;
                        } catch (err) {
                            throw new Error(err.message);
                        }
                    };

                    console.log(data);
                    ```
                </>
            </Tab>
            <Tab>
                <>
                    ```python
                    import requests
                    import os
                    from dotenv import load_dotenv

                    load_dotenv()

                    os.getenv('API_KEY')

                    url = "https://api.anify.tv/genres?apikey={}".format(os.getenv('API_KEY'))
                    response = requests.get(url)
                    data = response.json()

                    print(data)
                    ```
                </>
