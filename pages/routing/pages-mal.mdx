# Pages (MAL)
Used for querying page links for manga or text content for novels but using MAL IDs rather than AniList IDs. Technical details regarding the usage of the pages MAL function for manga/novels can be found below. Example code is provided for both JavaScript and Python, along with a response schema.

## Rate Limits
The pages MAL route has a rate limit of 60 requests per minute unless your API key is whitelisted.

## Route Schema (URL)
<Tabs items={["Parameters", "Query", "Post"]}>
    <Tab>
        <>
            ```
            https://api.anify.tv/pages-mal?providerId={providerId}&readId={readId}&apikey={apikey}
            ```
        </>
| Parameter |  Type  | Description                                                       | Required? | Default |
| :-------: | :----: | ----------------------------------------------------------------- | :-------: | :-----: |
|   providerId   | string | The provider ID of the manga/novel you're fetching pages for. Ex. `mangadex`. Can be obtained from `/chapters`. |    Yes    |  `""`   |
|   readId    | string | The read ID of the manga/novel you're fetching pages for. Can be obtained from `/chapters` (ID field). |    Yes    |  `""`   |
|   apikey  | string | Your API key |    Yes    |  `""`   |

        ## Request Samples

        <Tabs items={["JavaScript", "Python"]}>
            <Tab>
                <>
                    ```js
                    import dotenv from "dotenv";
                    dotenv.config();

                    import axios from "axios";

                    const apiKey = process.env.API_KEY ?? "";
                    ```
                    ```js
                    const url = `https://api.anify.tv/pages-mal?apikey=${apiKey}`;
                    const data = async () => {
                        try {
                            const { data } = await axios.post(url, {
                                providerId: "mangadex",
                                watchId: "d0c34c6a-9a5d-4783-b5a3-dc3d11799de7"
                            });
                            return data;
                        } catch (err) {
                            throw new Error(err.message);
                        }
                    };

                    console.log(data);
                    ```
                </>
            </Tab>
            <Tab>
                <>
                    ```python
                    import requests
                    import os
                    from dotenv import load_dotenv

                    load_dotenv()

                    os.getenv('API_KEY')

                    url = "https://api.anify.tv/pages-mal?apikey={}".format(os.getenv('API_KEY'))
                    response = requests.post(url, json={
                        "providerId": "mangadex",
                        "watchId": "d0c34c6a-9a5d-4783-b5a3-dc3d11799de7"
                    })
                    ```
