# Episode Covers (MAL)
Used for querying episode covers for a media but using MAL IDs rather than AniList IDs. Technical details regarding the usage of the episode covers MAL function for anime can be found below. Example code is provided for both JavaScript and Python, along with a response schema.

## Rate Limits
The episode covers MAL route has a rate limit of 40 requests per minute unless your API key is whitelisted.

        ## Request Samples

        <Tabs items={["JavaScript", "Python"]}>
            <Tab>
                <>
                    ```js
                    import dotenv from "dotenv";
                    dotenv.config();

                    import axios from "axios";

                    const apiKey = process.env.API_KEY ?? "";
                    const url = `https://api.anify.tv/episode-covers-mal?apikey=${apiKey}`;
                    const data = async () => {
                        try {
                            const { data } = await axios.post(url, {
                                id: "39535"
                            });
                            return data;
                        } catch (err) {
                            throw new Error(err.message);
                        }
                    ```
