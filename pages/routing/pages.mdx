# Pages
Used for querying page links for manga or text content for novels. Technical details regarding the usage of the pages function for manga/novels can be found below. Example code is provided for both JavaScript and Python, along with a response schema.

## Rate Limits
The pages route has a rate limit of 60 requests per minute unless your API key is whitelisted.

## Route Schema (URL)
<Tabs items={["Parameters", "Query", "Post"]}>
    <Tab>
        <>
            ```
            https://api.anify.tv/pages/{id}/{chapterNumber}/{providerId}/{readId}
            ```
        </>
| Parameter |  Type  | Description                                                       | Required? | Default |
| :-------: | :----: | ----------------------------------------------------------------- | :-------: | :-----: |
|   id    | string | The media ID of the manga/novel you're fetching pages for. |    Yes    |  `""`   |
|   chapterNumber    | number | The chapter number of the manga/novel you're fetching pages for. Can be obtained from `/chapters` (number field). |    Yes    |  `""`   |
|   providerId   | string | The provider ID of the manga/novel you're fetching pages for. Ex. `mangadex`. Can be obtained from `/chapters`. |    Yes    |  `""`   |
|   readId    | string | The read ID of the manga/novel you're fetching pages for. Can be obtained from `/chapters` (ID field). |    Yes    |  `""`   |
|   apikey  | string | Whitelisted API key |    No    |  `""`   |

        ## Request Samples

        <Tabs items={["JavaScript", "Python"]}>
            <Tab>
                <>
                    ```js
                    import axios from "axios";

                    const url = `https://api.anify.tv/pages`;
                    const data = async () => {
                        try {
                            const { data } = await axios.post(url, {
                                id: "801513ba-a712-498c-8f57-cae55b38cc92",
                                providerId: "mangasee",
                                chapterNumber: 384,
                                readId: "/read-online/Berserk-chapter-374"
                            });
                            return data;
                        } catch (err) {
                            throw new Error(err.message);
                        }
                    };

                    console.log(data);
                    ```
                </>
            </Tab>

## Response Schema

<Tabs items={["200 OK", "400 Bad Request", "500 Internal Server Error"]}>
    <Tab>
        **MIME Type:** `application/json`
        **Reason:** `Occurs when the request was successful.`
        ```json copy=false
        [
            {
                "url": "string",
                "index": "number",
                "headers": { [key: string]: string };
            }
        ]
        ```
    </Tab>
    <Tab>
        **MIME Type:** `application/json`<br />
        **Reason:** `Occurs when the the "id" parameter, "chapterNumber" parameter, "providerId" parameter, or "readId" parameter aren't given.`
        ```json copy=false
        { "error": "No ID provided." }
        { "error": "No chapter number provided." }
        { "error": "No provider ID provided." }
        { "error": "No read ID provided." }
        ```
    </Tab>
