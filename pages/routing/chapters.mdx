# Chapters
Used for fetching chapters for a given media. Technical details regarding the usage of the chapters function can be found below. Example code is provided for both JavaScript and Python, along with a response schema.

## Rate Limits
The chapters route has a rate limit of 40 requests per minute unless your API key is whitelisted.

        ## Request Samples

        <Tabs items={["JavaScript", "Python"]}>
            <Tab>
                <>
                    ```js
                    import axios from "axios";

                    const url = `https://api.anify.tv/chapters?apikey=${apiKey}`;
                    const data = async () => {
                        try {
                            const { data } = await axios.post(url, {
                                id: "a25e46ec-30f7-4db6-89df-cacbc1d9a900"
                            });
                            return data;
                        } catch (err) {
                            throw new Error(err.message);
                        }
                    };

                    ```
                </>
