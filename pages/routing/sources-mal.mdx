# Sources (MAL)
Used for querying m3u8 sources for a media but using MAL IDs rather than AniList IDs. Technical details regarding the usage of the sources MAL function for anime can be found below. Example code is provided for both JavaScript and Python, along with a response schema.

## Rate Limits
The sources MAL route has a rate limit of 60 requests per minute unless your API key is whitelisted.

## Route Schema (URL)
<Tabs items={["Query", "Post"]}>
    <Tab>
        <>
            ```
            https://api.anify.tv/sources-mal?providerId={providerId}&watchId={watchId}&episode={episode}&id={id}&subType={subType}&server={server}&apikey={apikey}
            ```
        </>
| Parameter |  Type  | Description                                                       | Required? | Default |
| :-------: | :----: | ----------------------------------------------------------------- | :-------: | :-----: |
|   providerId   | string | The provider ID of the show you're fetching sources for. Ex. `zoro`. Can be obtained from `/episodes`. |    Yes    |  `""`   |
|   watchId    | string | The watch ID of the show you're fetching sources for. Can be obtained from `/episodes` (ID field). |    Yes    |  `""`   |
|   episode    | number | The episode of the show you're fetching sources for. Can be obtained from `/episodes` (number field). |    Yes    |  `""`   |
|   id    | string | The MAL ID of the show you're fetching sources for. |    Yes    |  `""`   |
|   subType    | string | Must be either `sub` or `dub`. |    No    |  `"sub"`   |
|   server    | string | Streaming server of provider. Ex. `vidstream`. |    No    |  `"default streaming server for provider"`   |
|   apikey  | string | Your API key |    Yes    |  `""`   |

        ## Request Samples

        <Tabs items={["JavaScript", "Python"]}>
            <Tab>
                <>
                    ```js
                    import dotenv from "dotenv";
                    dotenv.config();

                    import axios from "axios";

                    const apiKey = process.env.API_KEY ?? "";
                    const url = `https://api.anify.tv/sources-mal?apikey=${apiKey}`;
                    const data = async () => {
                        try {
                            const { data } = await axios.post(url, {
                                providerId: "gogoanime",
                                watchId: "/kubo-san-wa-mob-wo-yurusanai-episode-10",
                                episode: 10,
                                id: "51815",
                                subType: "sub",
                                server: "gogocdn"
                            });
                            return data;
                        } catch (err) {
                            throw new Error(err.message);
                        }
                    };

                    console.log(data);
                    ```
                </>
            </Tab>
            <Tab>
                <>
                    ```python
                    import requests
                    import os
                    from dotenv import load_dotenv

                    load_dotenv()

                    os.getenv('API_KEY')

                    url = "https://api.anify.tv/sources-mal?apikey={}".format(os.getenv('API_KEY'))
                    response = requests.post(url, json={
                        "providerId": "gogoanime",
                        "watchId": "/kubo-san-wa-mob-wo-yurusanai-episode-10",
                        "episode": 10,
                        "id": "51815",
                        "subType": "sub",
                        "server": "gogocdn"
                    })
                    data = response.json()

                    print(data)
                    ```
                </>
