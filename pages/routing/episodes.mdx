# Episodes
Used for fetching episodes for a given media. Technical details regarding the usage of the episodes function can be found below. Example code is provided for both JavaScript and Python, along with a response schema.

## Rate Limits
The episodes route has a rate limit of 40 requests per minute unless your API key is whitelisted.

        ## Request Samples

        <Tabs items={["JavaScript", "Python"]}>
            <Tab>
                <>
                    ```js
                    import dotenv from "dotenv";
                    dotenv.config();

                    import axios from "axios";

                    const apiKey = process.env.API_KEY ?? "";
                    const url = `https://api.anify.tv/episodes?apikey=${apiKey}`;
                    const data = async () => {
                        try {
                            const { data } = await axios.post(url, {
                                id: "21"
                            });
                            return data;
                        } catch (err) {
                            throw new Error(err.message);
                        }
                    };

                    console.log(data);
                    ```
                </>
            </Tab>
            <Tab>
                <>
                    ```python
                    import requests
                    import os
                    from dotenv import load_dotenv

                    load_dotenv()

                    os.getenv('API_KEY')

                    url = "https://api.anify.tv/episodes?apikey={}".format(os.getenv('API_KEY'))
                    response = requests.post(url, json={'id': "21"})
                    data = response.json()

                    print(data)
                    ```
                </>
            </Tab>
